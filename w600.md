# W600

I personally own 7 of the [ThingsTurn TB-01 Devkit](https://github.com/sammothxc/w600-documentation/tree/main/tb-01-devkit-master) boards.

## PlatformIO

Create the folder `%HOMEPATH%/.platformio/boards` if it doesn't exist already. [Download your board file](https://github.com/sammothxc/platform-w60x/blob/master/boards) and put it in the boards folder.

Put this in your `platformio.ini`

```
[env:w600]
platform = 
framework = arduino
; framework = wm60x-sdk 
board = tb_01
```
